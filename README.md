# KlpMine

KlpMine — voxel sandbox на Rust и Bevy. Проект сделан как основа для собственной блочной игры: мир из чанков, управление от первого лица, разрушение и установка блоков, хотбар, предметы в мире, падение блоков и простая физика игрока.

Код разделён по системам: камера, мир, генерация, блоки, инвентарь, здоровье, настройки, рука игрока, отладка и ресурсы. Это позволяет добавлять новые механики отдельными модулями и не превращать проект в один большой файл.

## Запуск

```powershell
cargo run
```

## Управление

- `W A S D` — ходьба
- мышь — обзор
- `Space` — прыжок
- `Left Control` — бег
- `Left Shift` — приседание
- левая кнопка мыши — ломать блок
- правая кнопка мыши — поставить блок
- `Q` — выбросить выбранный блок
- колесо мыши или `1`-`9` — выбрать слот хотбара
- `F3` — отладочная информация
- `Esc` — настройки и выход

## Мир

Мир генерируется чанками. В генерации есть трава, земля, камень и песок. Камень лежит глубже, земля находится под поверхностью, трава покрывает верхний слой, а песок появляется низкими участками и падает вниз, если под ним нет опоры.

У игрока есть здоровье. Урон появляется при падении с высоты и от падающих блоков. После смерти предметы выпадают в мир, а игрок может возродиться через кнопку `Respawn`.
