# KlpMine

**KlpMine** — это voxel sandbox, вдохновлённый Minecraft, написанный на языке программирования **Rust** с использованием игрового движка **Bevy**. Проект создаётся как основа для собственной блочной игры с упором на производительность, модульность и возможность расширения.

Сейчас в игре уже реализованы генерация мира из чанков, разрушение и установка блоков, инвентарь и хотбар, простая физика игрока, выпадение предметов, система здоровья и другие базовые механики выживания.

## Запуск

Вы можете собрать проект самостоятельно:

```powershell
cargo run
```

или скачать последнюю готовую версию из [Releases](https://github.com/AmKilopa/KlpMine/releases).

Готовые сборки позволяют запустить игру без необходимости собирать проект вручную.
