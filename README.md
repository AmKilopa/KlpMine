# KlpMine

KlpMine — voxel sandbox на Rust и Bevy. Проект задуман как собственная блочная игра с миром из чанков, управлением от первого лица и архитектурой, которую можно развивать без переписывания основы.

Код разделен по зонам ответственности: запуск приложения, камера, отладочная информация, мир, блоки, генерация и построение мешей. Такой подход нужен, чтобы новые механики добавлялись отдельными системами и не превращали проект в один большой файл.

## Запуск

```powershell
cargo run
```

## Управление

- `W A S D` — ходьба
- мышь — обзор
- `Space` — прыжок
- `Left Control` — бег
- `Left Shift` — приседание
- удерживать левую кнопку мыши — сломать выбранный блок
- правая кнопка мыши — поставить выбранный блок
- `Q` — выбросить выбранный блок
- колесо мыши или `1`-`9` — выбрать слот хотбара
- `F3` — debug-меню
- `Esc` — меню настроек и выход

Звуки и текстуры проекта сделаны отдельно для KlpMine и лежат в `assets`.

## Ветки

- `main` — стабильное состояние
- `develop` — основная разработка
- `feature/...` — отдельные изменения
